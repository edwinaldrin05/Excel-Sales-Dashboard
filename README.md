# Excel Sales Dashboard

## Project Overview
This project is an interactive Sales Dashboard built using Microsoft Excel. It provides insights into sales performance using Pivot Tables, Pivot Charts, KPIs, and Slicers.

## Features
- Total Sales KPI
- Total Profit KPI
- Sales by Product
- Sales by Year
- Top 10 Customers by Sales
- Interactive Slicers
  - Year
  - Product
  - Country
  - City
  - Customer

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
